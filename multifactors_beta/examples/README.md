# 示例和教程

本目录包含多因子系统的完整使用示例和教程，帮助开发者快速上手。

## 📁 目录结构

### factor_examples/ - 因子计算示例
- `generate_bp_factor.py` - BP因子计算示例
- `calculate_custom_factor_example.py` - 自定义因子计算
- `quick_custom_factor_dev.py` - 快速因子开发

### workflows/ - 完整工作流示例  
- `simple_custom_factor_workflow.py` - 简单自定义因子工作流
- `standard_factor_workflow.py` - 标准因子开发工作流
- `correct_factor_workflow.py` - 正确的因子计算流程

### benchmarks/ - 性能基准测试
- `momentum_performance_benchmark.py` - 动量因子性能基准
- `technical_factors_performance_benchmark.py` - 技术因子性能基准

### demos/ - 功能演示
- `demo_claude_usage.py` - Claude Code使用演示
- `demo_agent_usage.py` - 智能代理使用演示

## 🚀 快速开始

### 1. 运行基础示例
```bash
# 查看BP因子计算示例
python examples/factor_examples/generate_bp_factor.py

# 学习标准工作流
python examples/workflows/standard_factor_workflow.py
```

### 2. 性能测试
```bash
# 运行动量因子性能基准
python examples/benchmarks/momentum_performance_benchmark.py
```

### 3. 功能演示
```bash
# 体验Claude使用演示
python examples/demos/demo_claude_usage.py
```

## 💡 学习路径

1. **初学者**: 从`factor_examples/`开始，了解基本因子计算
2. **进阶用户**: 学习`workflows/`中的完整工作流程
3. **性能优化**: 参考`benchmarks/`中的性能基准测试
4. **高级功能**: 体验`demos/`中的智能功能演示
