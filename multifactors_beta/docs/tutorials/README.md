### 学习建议

1. **按顺序学习**: 教程有编号，建议按顺序完成
2. **动手实践**: 跟着教程一步步操作，不要只看不做
3. **理解原理**: 不仅要知道怎么做，还要理解为什么这样做
4. **遇到问题**: 先查看[故障排查指南](../how-to/troubleshooting/)，再提问
