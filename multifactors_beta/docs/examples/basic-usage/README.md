## 📝 示例代码规范

每个示例文件应包含：

```python
"""
示例名称 - 简短描述

功能说明：
- 功能1
- 功能2

依赖：
- pandas
- numpy
- factors

运行方式：
python examples/basic-usage/example_name.py
"""

# 导入依赖
import pandas as pd
from factors.generators import calculate_ttm

# 配置参数
CONFIG = {
    'data_path': 'path/to/data',
    'output_path': 'path/to/output',
}

def main():
    """主函数"""
    print("开始执行...")

    # 1. 加载数据
    data = pd.read_pickle(CONFIG['data_path'])
    print("✓ 数据加载完成")

    # 2. 处理数据
    result = calculate_ttm(data)
    print("✓ 数据处理完成")

    # 3. 保存结果
    result.to_pickle(CONFIG['output_path'])
    print("✓ 结果保存完成")

    print("全部完成！")

if __name__ == '__main__':
    main()
```
