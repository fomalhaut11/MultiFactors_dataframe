## 🎯 如何使用操作指南

### 查找方式

1. **按分类查找**: 根据问题类型在上面的分类中查找
2. **搜索关键词**: 使用编辑器搜索功能查找相关文档
3. **查看索引**: 每个子目录都有详细的README索引

### 使用建议

- **明确目标**: 先确定你要完成什么任务
- **直接执行**: 操作指南提供具体步骤，可以直接照做
- **理解原理**: 如果想深入理解，查看[概念说明](../explanation/)
- **举一反三**: 理解步骤背后的逻辑，应用到类似问题

## 💡 贡献操作指南

欢迎贡献新的操作指南！请遵循以下原则：

1. **目标明确**: 标题清楚说明要解决什么问题
2. **步骤清晰**: 提供具体的、可执行的步骤
3. **简洁实用**: 直奔主题，不讲不必要的背景
4. **经过验证**: 步骤必须经过实际验证可行

操作指南模板：

```markdown
# 如何做XXX

**目标**: 明确说明要完成什么
**前置条件**: 需要什么准备工作
**预计时间**: XX分钟

## 步骤

1. 第一步...
2. 第二步...
3. 第三步...

## 验证

如何验证操作成功

## 故障排查

常见问题及解决方法

## 相关文档

- [相关概念](../explanation/...)
- [相关教程](../tutorials/...)
```
