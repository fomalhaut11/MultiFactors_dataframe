## 🔍 故障排查流程

### 通用排查步骤
1. **确认错误信息** - 记录完整的错误信息和堆栈
2. **检查日志文件** - 查看相关日志获取更多信息
3. **验证环境配置** - 确认Python版本、依赖包版本
4. **检查数据完整性** - 验证输入数据是否正确
5. **查阅文档** - 查看相关功能文档
6. **搜索Issue** - 在GitHub Issues中搜索类似问题
